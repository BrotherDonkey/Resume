Arda Boards

(In progress) A topic and comment site for fans of J.R.R. Tolkien and his work. Tools Used: ExpressJS for server, routing, and RESTful API for comments. MongoDB for storage, Pug (Jade) for HTML templates, and AngularJS for responsive front end, Bootstrap for styling.

The Coalition of Refugees from Burma

(Currently a work in progress) A pro-bono project for a local non-profit that educates, supports, and assists refugees now living in Seattle. This site is made with Jekyll, JavaScript, a pinch of jQuery, and love.

A website for my novels
